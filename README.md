# 💤 LazyVim Setup

Welcome to my personal **LazyVim** configuration!  
This setup is beginner-friendly, snappy, and perfect for daily development in WSL with transparency support.

---

## ✨ Features

- ⚡ **Blazing fast startup**
- 🧠 **Intuitive keybindings** with `<leader>` as `<Space>`
- 🔍 Built-in **Telescope fuzzy finder**
- 🚀 **LSP support**, Autocompletion, Snippets
- 📁 Custom folders like `~/Dev/dsa` for DSA practice
- 🔖 **Harpoon** for quick navigation
- 🌈 Matching terminal & Neovim theme (Tokyo Night with opacity)
- 🪟 Seamless clipboard support between WSL and Windows

## 📦 Requirements

- Neovim `v0.10+`
- `ripgrep`, `fd`, `fzf`, `bat`, `lazygit`, `zoxide` (optional but recommended)
- WSL 2 (for Windows users)
- [wezterm](https://wezfurlong.org/wezterm/) (optional terminal)

## 🛠 Installation

git clone https://github.com/sanju2op/lazyvim-config.git ~/.config/nvim

Then launch Neovim:

nvim

> 📦 LazyVim will auto-install everything it needs on first run.
